# 昇腾310B手势识别系统——项目功能报告书

## 1. 系统概述

本项目基于昇腾310B（OrangePi AIpro）AI处理器和STM32F103C8T6微控制器，设计并实现了一套实时手势识别控制系统。USB摄像头采集画面送入昇腾310B进行YOLOv10手势检测，检测画面通过OpenCV在本地显示器实时显示，检测结果经GestureDebouncer消抖后通过USB CDC虚拟串口发送至STM32，STM32解析后输出PWM信号控制风扇转速与舵机角度。

```
┌──────────┐     ┌──────────────┐     ┌──────────────┐
│   USB    │────▶│  昇腾310B    │────▶│ STM32F103    │
│  摄像头  │     │  NPU推理     │     │  PWM控制     │
└──────────┘     └──────────────┘     └──────┬───────┘
                                             │
                                      ┌──────┴───────┐
                                      │              │
                                 ┌────▼────┐   ┌────▼────┐
                                 │ SG90×2  │   │ L298N   │
                                 │ 摆风舵机│   │ 风扇驱动│
                                 └─────────┘   └─────────┘
```

## 2. 功能列表

本系统实现的核心功能如下：

| 序号 | 功能 | 说明 |
|------|------|------|
| 1 | 手势检测与识别 | YOLOv10n_gestures模型在昇腾310B NPU上实时检测34类手势 |
| 2 | 检测画面本地显示 | OpenCV在本地显示器实时展示检测框、类别标签与置信度 |
| 3 | GestureDebouncer消抖 | 三帧确认+去重+消失滞后，误触发率从~8%降至<1% |
| 4 | 风扇PWM调速 | 4档风速（CCR=300/500/700/900），通过L298N驱动 |
| 5 | 系统开关机 | palm/stop开机，fist全关机（风扇停+舵机回中） |
| 6 | 左右摆风 | ok启动/timeout停止，预判式折返算法，SG90舵机 |
| 7 | 上下摆风 | like启动/dislike停止，SG90舵机 |
| 8 | 状态指示 | 板载PC13 LED收到手势数据时闪烁 |
| 9 | 串口通信 | USB CDC虚拟串口，换行分隔JSON协议 |

## 4. 功能详细说明

### 4.1 手势检测与识别

- **模型：** YOLOv10n_gestures，支持HaGRIDv2数据集定义的34类手势
- **推理平台：** 昇腾310B NPU（OrangePi AIpro），CANN 8.5 + PyACL
- **输入：** USB摄像头 640×480，letterbox缩放至640×640
- **输出：** 检测框坐标（xyxy）、类别ID（0-33）、置信度分数（0-1）
- **显示：** OpenCV在本地显示器实时叠加绘制检测框与标签

**支持的34类手势：**

| ID | 手势名称 | ID | 手势名称 | ID | 手势名称 |
|----|----------|----|----------|----|----------|
| 0 | grabbing | 12 | take_picture | 24 | stop_inverted |
| 1 | grip | 13 | dislike | 25 | three |
| 2 | holy | 14 | fist | 26 | three2 |
| 3 | point | 15 | four | 27 | two_up |
| 4 | call | 16 | like | 28 | two_up_inverted |
| 5 | three3 | 17 | mute | 29 | three_gun |
| 6 | timeout | 18 | ok | 30 | thumb_index |
| 7 | xsign | 19 | one | 31 | thumb_index2 |
| 8 | hand_heart | 20 | palm | 32 | no_gesture |
| 9 | hand_heart2 | 21 | peace | 33 | — |
| 10 | little_finger | 22 | peace_inverted | — | — |
| 11 | middle_finger | 23 | rock | — | — |

### 4.2 GestureDebouncer消抖传输

针对YOLOv10连续帧推理中的**单帧假阳性**和**类别抖动**问题，设计三级消抖策略：

| 策略 | 机制 | 效果 |
|------|------|------|
| 三帧确认窗口 | 滑动窗口长度3，同一手势≥2帧才确认发送 | 过滤单帧假阳性 |
| 确认后去重 | 记录最后发送ID，相同手势不重复发送 | 减少不必要的PWM更新 |
| 手势消失滞后 | 连续5帧无检测才确认消失 | 避免短暂遮挡导致控制中断 |

### 4.3 风扇PWM调速

- **驱动方式：** STM32 TIM3_CH1（PA6）输出1kHz PWM → L298N ENA
- **控制逻辑：** L298N IN1固定高、IN2固定低（正转）
- **档位设计：**

| 档位 | CCR值 | 触发手势 |
|------|-------|----------|
| 开机怠速 | 300 | palm (20) / stop (24) |
| 1档（低） | 300 | one (19) |
| 2档（中低）| 500 | peace (21) 等 |
| 3档（中高）| 700 | three (26) 等 |
| 4档（高） | 900 | four (15) |
| 关机 | 0 | fist (14) |

### 4.4 舵机摆风控制

- **硬件：** SG90舵机×2，TIM4_CH1（PB6）左右、TIM4_CH2（PB7）上下
- **PWM参数：** 50Hz，CCR范围 200-700（对应约90°扫描范围）
- **预判式折返算法：** 每10ms更新一次，步长2。更新前预先计算next位置，若预判越界则在本周期提前折返并将CCR强制弹离边界内侧2步长，从原理上消除端点停顿与抖动
- **独立计时：** 两路舵机使用独立时间戳变量，初始错开5ms避免同时跳变

### 4.5 系统状态机

```
                    palm/stop
    ┌──────────────────────────┐
    │                          │
    ▼                          │
┌───────┐     fist      ┌───────┐
│ OFF   │──────────────▶│  ON   │
│ 关机态│               │ 开机态│
└───────┘◀──────────────└───────┘
             开机后可执行:
             • 风扇调速 (1-4档)
             • 左右摆风 开/关
             • 上下摆风 开/关
```

- palm/stop → 开机（system_on = 1），任何状态可触发
- fist → 关机（system_on = 0，风扇停+舵机回中），任何状态可触发
- 其他控制指令仅在开机态生效

### 4.6 串口通信协议

- **物理层：** USB CDC虚拟串口，波特率115200
- **数据格式：** 每行一个JSON对象，`\n` 作为帧分隔符

```json
{"c": 16, "n": "like", "s": 0.92, "x": 320, "y": 240, "w": 120, "h": 150}
```

| 字段 | 含义 | 类型 |
|------|------|------|
| c | 类别ID（0-33） | int |
| n | 手势名称 | string |
| s | 置信度（0-1），<0.5不发送 | float |
| x, y | 边界框左上角坐标 | int |
| w, h | 边界框宽高 | int |

## 6. 硬件连接

| STM32引脚 | 功能 | 连接目标 | 说明 |
|-----------|------|----------|------|
| PA6 | TIM3_CH1（PWM） | L298N ENA | 风扇转速PWM控制（1kHz） |
| PB3 | GPIO输出 | L298N IN1 | 正转控制（固定高电平） |
| PB4 | GPIO输出 | L298N IN2 | 反转控制（固定低电平） |
| PB6 | TIM4_CH1（PWM） | 舵机1信号线 | 左右摆风舵机（50Hz） |
| PB7 | TIM4_CH2（PWM） | 舵机2信号线 | 上下摆风舵机（50Hz） |
| USB_DP/DM | USB CDC | 昇腾310B USB口 | 虚拟串口通信 |
| PC13 | GPIO输出 | 板载LED | 手势接收指示 |
