# 基于昇腾310B的实时手势识别控制系统

> **课程名称**：2026年嵌入式课设  
> **题目编号**：题目6  
> **提交日期**：2026-06-30

## 1. 项目介绍
本项目是一个基于 **昇腾310B (Orange Pi AIpro)** 的端侧边缘AI系统。系统利用 **YOLOv10** 目标检测模型（基于HaGRIDv2数据集预训练，支持34类手势）在NPU上进行实时推理。

核心功能包括：
- **视觉识别**：实时检测并分类手势（如点赞、握拳、掌心等）。
- **硬件控制**：识别结果通过串口发送至 **STM32F103C8T6**，驱动小风扇实现无接触式控制（加速、减速、停机、反转）。
- **远程监控**：通过 WebRTC 技术将带有检测框的视频流推送到浏览器端。

本系统完整覆盖了从 **PyTorch模型导出ONNX → ATC转换OM → PyACL推理 → 业务逻辑处理** 的边缘AI部署全链路。

## 2. 基本原理
1.  **AI推理**：使用YOLOv10的NMS-free检测头设计，输入640×640图像经Letterbox预处理后送入NPU。检测结果映射回原分辨率并叠加显示。
2.  **通信与控制**：Python端通过串口（UART）发送指令，STM32接收指令后查表映射，输出不同占空比的PWM波控制L298N/TB6612驱动模块，从而调节直流电机转速。
3.  **视频推流**：利用CANN VENC硬件编码为H.264格式，结合 `aiortc` 库实现低延迟WebRTC推流。
